# ALL_ABOUT_JAVA

A collection of Java programs covering core programming concepts, Object-Oriented Programming (OOP), exception handling, collections, file handling, and practice exercises. This repository was created to strengthen Java fundamentals through hands-on coding.

---

##  Topics Covered

###  Java Basics
- Variables & Data Types
- Operators
- User Input
- Loops
- Conditional Statements (if-else, switch)
- Functions/Methods

###  Object-Oriented Programming (OOP)
- Classes & Objects
- Constructors
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Access Modifiers

###  Advanced Java
- Exception Handling
- Collections Framework
- File Handling
- Functions & Methods
