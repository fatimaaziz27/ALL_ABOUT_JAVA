# ALL_ABOUT_JAVA

A collection of Java practice programs covering loops, conditionals, arrays, lists, strings, number operations, and sorting.
This repository contains 20+ beginner-to-intermediate Java exercises, all written in a single file (ALL_ABOUT_JAVA.java) for easy learning and practice.
